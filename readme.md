# get-emails-cli

> Get all emails in a text file or stdin

## Install

```
$ npm install --global get-emails-cli
```

## Usage

```
$ get-emails --help

  Usage
    $ get-emails <file>
    $ cat <file> | get-emails

  Example
    $ get-emails file.txt
    sindresorhus@gmail.com
    unicorn@rainbow.com
```
